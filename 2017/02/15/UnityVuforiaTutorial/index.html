Setting up Vuforia in Unity to Develop Your First AR Program

2017-02-15

This is a step-by-step tutorial for beginners to configure Vuforia in Unity and to use basic prefabs to create a simple AR test program.

This tutorial has four parts:

  1. Setting up Vuforia account and image target database that used to recognise and augment on
  2. Configurations in Unity project to work with Vuforia
  3. Design a simple scene to test the previous settings
  4. How to export the Unity project to iOS/Android devices

At the end of this tutorial, some learning resources will be provided for you to get familiar with the Unity developing environment, along with some websites and tools that you can use to find or develop character models and animations.

0. Prerequisites

  • macOS Sierra Version 10.12.3
  • Unity 5.5.0f3 Personal (64bit) (Personal Plan) https://store.unity.com

Part 1: Download Vuforia and register for a developer account

  1. Open the Vuforia developer page: https://developer.vuforia.com, register and log in.
  2. Click on Download and find the SDK for Unity vuforia-unity-6-2-6.unitypackage (46.20 MB).
  3. Click on Develop, under the License Manager tab, click on the Add License Key button and generate a key for your project. Choose Development as the project type.
  4. Click on Target Manager tab, click on the Add Database button to create a image target package for your project. Leave the Device type as default.
  5. Click on the database you created, add image(s) to use as the target(s) to recognise. If you do not have any specific images at the moment, I would recommend using a QR code (such as your WeChat code) as it is handy, has enough features and is easy for the following tests. Feel free to add more than one target if you find it necessary for your project. Try to get a five star rating for your image target because it will make the tracking more precise and stable. You can see the image features by clicking on the image you uploaded and check the Show Features.
  6. Once the image target database is created, click on Download Database (All) and choose Unity Editor. You will get a .unitypackage file.

Part 2: Create an Unity project and set the environment

  1. Open Unity, create a 3D project. You can use the default layout or change the layout by clicking on the drop-down menu at the top right corner. For this tutorial I will keep the default layout, although I usually choose the 2 by 3 layout and drag the Project window under the Hierachy window. You can explore later and find the layout that makes you comfortable.
  2. Drag and drop the vuforia-unity-6-2-6.unitypackage to the Assets folder and import all. You may unselect some of the plugins if you are not going to use them. The packages can also be imported using the top bar menu by clicking on Assets -> Import package -> Custom package and selecting the package in your directory.
  3. Drag and drop the Vuforia image target package (UnityARDemo.unitypackage) to the Assets folder and import all.
  4. Delete the Main Camera in the scene, find the ARCamera prefab under Assets/Vuforia/Prefabs. You can also use keyword search to find it.
  5. Drag and drop the ARCamera prefab to the scene or to the hierarchy. Set it to position (0, 0, 0) by clicking on Reset in the transform settings drop-down menu.
  6. Open Vuforia configuration.
  7. Go to the Vuforia developer page, copy the license key you created and paste in the App License Key box. Load your image target dataset and activate the target(s).

Part 3: Simple AR program demo

  1. Cmd+S to save the scene so far.
  2. Find the ImageTarget prefab and drag it to the scene. Set it to (0, 0, 0).
  3. In the ImageTarget inspector, set the Database and Image Target.
  4. Create a something to augment. Here I use a simple cube to demonstrate. Create a cube by clicking on GameObject -> 3D Object -> Cube. In your project, you may want to use characters with animations or build your environment with multiple gameobjects.
  5. Resize the cube to make it smaller than the image target (you donâ€™t have to) and stand on top of it. Here I set the scale to (0.5, 0.5, 0.5) and the position to (0, 0.25, 0).
  6. Make the Cube a child of ImageTarget. Your first AR program is ready! Click on play and see how it goes.

  1. Mixamo and Adobe Fuse: https://www.mixamo.com
    You can find high-quality and free characters with many animations.     This website has all kinds of models. Some of them are well animated and rigged.
